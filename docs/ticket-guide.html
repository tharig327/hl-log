FloorSync – Ticket Submission Guide

How to fill out the Issue Tickets form for each issue type

Quick reference

The ticket form has three issue types — Applicator, Machine, and Setup Sheet — each with slightly different required fields. Select a tab below for a field-by-field walkthrough.

🔧
Applicator Issue
Use when the applicator itself has a problem — damage, wear, misalignment, etc.
Fields marked REQ must be filled in before submitting. Fields marked OPT are optional but helpful.
YOUR NAME REQ
First and last name of the person submitting the ticket — not the supervisor or engineer, whoever is filling out the form right now.
e.g. John Smith
CUSTOMER OPT
The customer whose job this applicator is running for at the time of the issue, if applicable.
e.g. BraunAbility
APPLICATOR # REQ
The applicator asset number. Must be in the format T- followed by exactly 4 digits.
T-0000 → T-9999
PRESS / MACHINE OPT
The press or machine the applicator is currently installed on.
e.g. KM-4T #2
ISSUE TYPE REQ
Select the best-matching issue from the dropdown. If nothing fits, choose Other — describe below and explain in the description box.
DESCRIPTION
Required if Other is selected. Be specific: describe what is happening, when it started, and any symptoms.
e.g. "Crimp height drifting high after ~200 cycles, adjusted shut height with no effect"
PRIORITY REQ
How urgent is this? See the priority guide below.
PHOTOS OPT
Attach photos of the applicator or defective crimps if available. Click or drag into the upload area.
Issue Type Options — Applicator
Crimp height out of spec Measured crimp height is outside the spec window; adjustment has not resolved it.
Pull force failure Terminal pull test results are below the minimum per USCAR-21 / IPC-WHMA-A-620 or HL internal spec.
Applicator damage / broken part Visible physical damage — cracked die holder, broken crimp tooling, bent feed mechanism, etc.
Feed / index issue Terminal strip not advancing correctly; skipping, jamming, or double-feeding.
Other — describe below None of the above apply. Use the description box to explain the issue in detail.
Priority Guide
High — affecting production now Production is stopped or parts are being scrapped. Needs immediate attention.
Medium — can finish current run Issue exists but can be managed through the current job. Needs repair before next setup.
Normal — can wait for scheduled repair Minor issue or observation that doesn't affect current output. Log for next PM or scheduled repair.
Tips — Applicator tickets
  • Always include the T-number — this is how the ticket gets routed to the right applicator record.
  • Include the press name so the technician knows where to find the applicator without hunting.
  • If you have crimp height measurements or pull test readings, put them in the description — it speeds up diagnosis significantly.
  • A photo of the defective crimp cross-section or the damaged component is always helpful.
⚙️
Machine Issue
Use when the press, crimper, or other equipment itself has a problem — not the applicator.
Fields marked REQ must be filled in before submitting. Fields marked OPT are optional but helpful.
YOUR NAME REQ
First and last name of the person submitting the ticket.
e.g. John Smith
CUSTOMER OPT
Customer job running at the time of the issue, if applicable.
e.g. BraunAbility
MACHINE # REQ
The press or machine identifier. Use the name/number posted on the machine.
e.g. KM-4T #2
ISSUE TYPE REQ
Select the best-matching issue from the dropdown.
PRIORITY REQ
Select how urgently this needs to be addressed. See the priority guide below.
PHOTOS OPT
Photos of the machine issue — error codes on the screen, damaged components, etc.
Machine vs. Applicator: If the press is running fine but the terminal results are bad, the issue is likely the applicator, not the machine. Submit an Applicator ticket instead. If the press itself is making noise, throwing errors, or won't cycle, submit a Machine ticket.
Issue Type Options — Machine
Won't cycle / no power Press will not actuate or has no power. Check E-stops first before submitting.
Error code / alarm Machine is displaying an error or alarm code. Include the code in the description if possible.
Abnormal noise or vibration Grinding, knocking, or vibration not normally present during operation.
Shut height / stroke issue Machine is not reaching or maintaining correct shut height consistently.
Pneumatic / air issue Low air pressure, leaks, or pneumatic components not operating correctly.
Other — describe below None of the above. Describe the issue in detail.
Priority Guide
High — affecting production now Machine is down; production cannot continue.
Medium — can finish current run Issue is present but production can limp through the current job with caution.
Normal — can wait for scheduled repair Minor — log for next PM cycle or scheduled maintenance window.
Tips — Machine tickets
  • If the machine shows an error code, photograph the screen and attach it — this is the fastest path to diagnosis.
  • Note whether the issue is consistent or intermittent, and approximately how many cycles between failures if intermittent.
  • If you had to E-stop the machine, leave it in the state you stopped it if safe to do so — don't reset before the tech arrives.
📄
Setup Sheet Issue
Use when there's an error, discrepancy, or missing information in a crimp setup sheet or work instruction document.
Fields marked REQ must be filled in before submitting. Fields marked OPT are optional but helpful.
YOUR NAME REQ
First and last name of the person submitting the ticket.
e.g. John Smith
CUSTOMER OPT
Customer the setup sheet applies to, if applicable.
e.g. BraunAbility
TERMINAL # REQ
The terminal part number the setup sheet is for. Use the full manufacturer part number from the setup sheet header.
e.g. 02233
AWG OPT
Wire gauge the setup sheet covers, if the issue is specific to a particular wire size.
e.g. 18 AWG
ISSUE TYPE REQ
Select the type of document issue from the dropdown.
DOCUMENT / FILE REFERENCE OPT
The document number, file name, or revision of the setup sheet in question. Helps engineering locate the exact file.
e.g. SS-2024-0042, rev B
PRIORITY REQ
How urgently the document needs to be corrected or updated.
PHOTOS OPT
Photos of the specific section of the document with the error, or of actual vs. expected results on the floor.
Issue Type Options — Setup Sheet
Wrong crimp height / dimension The crimp height target, minimum, or maximum listed on the sheet does not match the validated spec.
Wrong wire range or AWG Wire gauge or cross-section listed on the document doesn't match what is being run or what the terminal supports.
Incorrect applicator or tooling number The applicator T-number or die/locator reference on the sheet points to the wrong tool.
Missing information A required field is blank — shut height, pull force, wire type, etc.
Document not found No setup sheet exists for this terminal / wire combination and one needs to be created.
Other — describe below None of the above. Describe the discrepancy in detail.
Priority Guide
High Error is causing active production defects or an operator cannot run without a corrected document.
Normal Discrepancy found during review; not blocking current production. Update at next revision cycle.
Tips — Setup Sheet tickets
  • Include both the document reference number AND the terminal part number — searching by either should pull up the same file in engineering.
  • If the sheet has the wrong crimp height, include what the sheet says AND what the correct value should be (from a validated run or master spec).
  • A photo of the wrong section circled is faster than any written description.
  • If the document simply doesn't exist yet, set priority based on when that terminal is next scheduled to run.
